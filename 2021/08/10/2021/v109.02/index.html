原文 : TERA 테라 | BLUEHOLE

Update

1. 共鸣副本

  • 为确保系统稳定性，共鸣副本更新已推迟至8月25日。我们将尽最大努力向您展示更流畅的游戏环境和完美的面貌

  • 共鸣副本位于巴德拉特别地区

    • 只有在巴德拉特别地区的巴德拉村庄通过副本传送石才能进入共鸣副本

    • Lv70 可以通过完成“[重要] 探索共鸣副本”任务来进入共鸣副本

    • 共鸣之门有两种类型：“共鸣之门”（左）和“血染的共鸣之门”（右）。 （通过不同的大门进入的地下城也各不相同）

    • “血染的共鸣之门”是为以最高级别为目标的守护者提供的 （部分奖励无法通过‘共鸣之门’获得的）

      • 每个共鸣之门进入的副本、等级和推荐装备等级有所不同
      门 共鸣之门 血染的共鸣之门
      阶段 1~3阶段 1~n阶段
      副本名称 分裂的核心融合所 不灭的凯尔塞克隐藏地
      装备等级 504 510

    • 单击共振门（或按快捷键 F），则显示该窗口

      • 单击弹出窗口底部的传送移动到共鸣副本 （5人组队挑战）

    • 如果挑战成功，可以挑战的阶段会增加

      • 入场需要一个共鸣石

    • 在Lv65以上的副本中击败BOSS后可以获得共鸣石

  • 共鸣石仅在本赛季期间可用。 （AWAKE 共鸣石 仅在 AWAKE 赛季可用）

  • 当你进入共鸣副本时，时间倒计时就开始了

    • 在规定时间内成功通关副本可以获得奖励，即使在规定时间内通关失败，也可以通过击败boss获得最低奖励
      • 共鸣地牢不提供普通掉落奖励，只有成功和失败两种奖励。
    • 超过时间限制之后，副本的共鸣效果会消失
    • 共鸣副本是随机分配的
    • 目前在血染的共鸣之门中，只有不灭的凯尔塞克隐藏地，以后会持续更新其他副本
    • 共鸣激活的效果是随机确定的

  • 在通过血染的共鸣之门进入的共鸣副本后，在规定的时间通关成功，可以获得次元碎片

    • 次元碎片刻印在装备上可以永久获得一些属性
    • 次元碎片只能用于本赛季装备

  • AWAKE次元碎片只能在AWAKE赛季进行中使用，AWAKE赛季结束后下个赛季不能再使用

  • 通过‘血染的共鸣之门’入场的共鸣副本，如果规定时间内成功通关的话，共鸣的祭坛就会被召唤出来，通过祭坛可以强化次元碎片

  • 次元碎片可以通过装备 > 锻造选项卡 > 共振刻印选项卡刻印在装备上 （归属于自己的设备）

2. 排行榜

  • 加入共鸣副本专属排行榜

    • 在已有的排行榜UI中添加[共鸣副本排名]标签页

    ​ 以上副本列表及排名奖励为任意设定值，实际更新时可能会有所变动

    ​ 后续更新后，您可以在游戏中查看详细的副本列表和名次奖励

    • 通过挑战血染的共鸣之门进入的共鸣副本后，您的排名将根据挑战难度和通关时间记录在排行榜上
    • 每次赛季变更时记录和排名会重置
    • 共鸣副本排名将按时间顺序排列
      • 副本难度越高，通关时间越快的记录将排列在前面

3. 装备和物品

  • 圣物/遗物改为不归属状态也可以升级

4. 用户界面和其他改进

  • 一个篝火被放置在巴德拉村庄

  • 游戏指南菜单的位置已从系统菜单的左侧更改为 Terra Shop 菜单的右侧

5. 战场

  • 调整了战场攻城武器的平衡性
    • 移动攻城武器
      • 移动速度降低
      • 伤害降低
    • 增加所有攻城武器的修理时间

    BUG修复

  • 던전 보스 몬스터의 시스템 데칼 범위와 실제 피격 범위가 상이한 문제

  • 바라카니아 구릉지 특정 지역의 그래픽이 깨지는 문제

  • 정령사로 플레이 중 간헐적으로 스킬, 채팅 등 사용이 불가능한 문제

  • 거래 중개소에 물품 등록 시 물품 수량 입력 란에 숫자 키가 동작하지 않는 문제

  • 다른 플레이어 이펙트 표시 옵션 비활성화 시 던전 패턴 이펙트가 보이지 않는 문제

  • 펑크MAX 소파 탈것 탑승 상태의 그래픽 품질이 어색한 문제

  • 파트너 툴팁의 8~10 등급의 교감도 상승 범위가 오 기재된 문제

  • 채팅 탭 옵션 ‘파티/공격대 구성 시 채팅 입력 채널 자동 변경’ 옵션이 항상 동작하는 문제

  • 아마룬의 지혜 (거래불가) 아이템이 사용 불가한 문제

  • 월광 무사 반월참 스킬 1회 사용 시 적월/개전으로 스킬 초기화가 불가능한 문제

  • 마공사 빔 판넬 소환 적용 상태에서 속사 시전 후 빔 판넬 소환 종료로 인해 속사 시전이 멈추는 문제

  • 리카노르 투기장 몬스터가 간헐적으로 타켓팅이 비정상적인 문제
